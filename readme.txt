Плагин автоматически создаёт файл YML(XML) для добавления сайта на WooCommerce в каталог Rozetka Marketplace.
Имеется выбор, какие продукты добавлять, а какие не добавлять в файл выгрузки и отчёт о работе.

== Description ==
Плагин имеет возможность:
- автоматическая генерация YML(XML) при обеновлении или добавлении продуктов.
- выбор продуктов для добавления (выборочно добавлять продукты или сразу все)
- настройка названия и логотипа сайта для Rozetka
- отчёт о том, какие родукты добавлены для Rozetka
- задание специальной цены для продукта, когда он добавлен в каталог Rozetka
